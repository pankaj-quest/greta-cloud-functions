# Greta Project

This project was generated and is managed by **Greta AI**.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Python/FastAPI (backend)
- MongoDB (database)

## Development

```sh
# Install dependencies
npm install

# Start development server
npm run dev
```

## Backend

The backend uses Python/FastAPI with MongoDB for data persistence.

```sh
cd backend
pip install -r requirements.txt
python server.py
```
